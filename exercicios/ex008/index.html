Principais formatações

Negrito / Destaque

Nesta frase temos um termo em negrito usando a tag <b>. (não semântica)

Nesta frase temos um termo em destaque usando a tag <strong>. (semântica)

Itálico / Ênfase

Nesta frase temos um termo em itálico usando a tag <i>. (não semântica)

Nesta frase temos um termo em ênfase usando a tag <em>. (semântica)

Texto marcado

Podemos criar também um texto marcado com a tag <mark>. (semântico)

Neste exemplo o seletor que utilizamos foi a tag <mark> que receberá as declarações das CSS. Utilizamos este seletor no <head> do HTML para mudar a cor de todos os marcadores de uma vez.
Ex:
<style>
mark {
background-color: aqua;
}
<style>

Texto grande e pequeno

A tag <big> já está obsoleta, mas era utilizada para deixar um texto grande. (não semântica)
Esta tag se tornou obsoleta pois é possível fazer esta alteração pelo CSS.

Já a tag <small> ainda é utilizada pois adiciona sentido a frase, é usada para deixar textos pequenos. (semãntica)

Texto deletado

Podemos marcar um texto como excluido para indicar que ele deve ser lido, mas não considerado. Utilizamos a tag <del>. (semântica)

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois, com a tag <ins>. (semântica)

Existe também o sublinhado com a tag <u>. (não semântica)

Texto sobrescrito

Para inserir coisas do tipo x20+3 utilize a tag <sup>. (semântica)

Texto subscrito

Para inserir coisas do tipo h20 utilize a tag <sub>. (semântica)
