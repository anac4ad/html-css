Parágrafos e Quebras de linhas

Você pode escrever um parágrafo de qualquer jeito. Basta colocar tudo no meio do par de tags <p> e </p>.

Se precisar quebrar o texto em algum lugar específico
como esse, você pode usar a tag <br>.

HTML Entities e Emojis

Para utilizar um caractere especial basta utilizar o & mais o HTML Entity.
Exemplo: & + copy = ©
Outros HTML Entities: € £ ¥ ®

Vamos agora adicionar alguns emojis.
Os emojis possuem um Unicode (Ex: U+1F604), para saber qual é o Unicode de um emoji basta pesquisar no site Emojipedia.
Para adicionar um emoji basta substituir o U+ por &#x.
Exemplos de emojis: 🔥 💣
