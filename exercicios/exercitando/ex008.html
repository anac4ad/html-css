Principais formatações

Destaque

Nesta frase temos um termo em destaque usando a tag <strong>. (semântica)

Ênfase

Nesta frase temos um termo em ênfase usando a tag <em>. (semântica)

Texto marcado

Podemos criar um texto marcado com a tag <mark>. (semântica)

Já neste exemplo utilizaremos a tag <mark> dentro da tag <style> do CSS, que se encontra no <head> do html.
Ex: <style>
mark {
background-color: aqua;
}
<style>

Texto pequeno

A tag <small> é utilizada para deixar textos pequenos. (semântica)

Texto deletado

Podemos marcar um texto como deletado para indicar que ele deve ser lido, mas não considerado. Utilizamos a tag <del>. (semântica)

Texto inserido

Podemos marcar um texto como inserido para ênfase e indicar que ele foi adicionado depois, com a tag <ins>. (semântica)

Texto sobrecrito e subscrito

Para inserir sinais como este x20+3 utilize a tag <sup>. (semântica)

E sinais como este h2O utilize a tag <sub>. (semântica)
